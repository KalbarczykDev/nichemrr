# Monetization Ideas

## Simple / Low Effort

- **Freemium seats** — free tier sees top 10 niches only, paid unlocks full list + deal scores. One Stripe integration, no infra changes.
- **One-time lifetime deal** — sell on your own or list on AppSumo. Works well for indie tools, no recurring billing complexity.
- **Affiliate links** — TrustMrr likely has an affiliate program. Add referral links to startup listings. Passive income.

## Medium Effort

- **Weekly email digest** — paid subscription gets a curated "top 5 niches this week" email. Resend already set up.
- **API access** — charge developers who want the scored/grouped data as JSON. API layer already exists.
- **Alerts** — paid users set a niche + MRR threshold and get emailed when a matching startup appears. High perceived value for acquisition hunters.

## Requires More Product Work

- **Deal room** — paid users save startups to a watchlist, add notes, track status. Turns it from a research tool into a workflow tool.
- **White-label reports** — export a niche analysis as a PDF. Charge per export or as a paid feature.

## Recommendation

Most natural fit given current users (SaaS acquisition hunters): **alerts + freemium**.
Someone willing to pay for deal alerts is already a high-intent buyer — highest conversion likelihood.
